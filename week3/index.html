Our Subjects

Front-end development

Front-end web development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly.

Learn Web Frameworks

Why Use One. There are a lot of good reasons to use a front-end framework instead of starting all your code from scratch: Save time! Obviously, if you're writing every single line of code by yourself, it's going to take a lot longer to launch your website.

Database

Database, also called electronic database, any collection of data, or information, that is specially organized for rapid search and retrieval by a computer.To access information from a database, you need a database management system (DBMS).

Learning Git and GitHub

Learn how to use version control in Git and Github with our online course in collaboration with Github. Learn how to effectively use version control as a web developer.

Back-End Development

A back-end web developer is responsible for server-side web application logic and integration of the work front-end developers do. Back-end developers are usually write the web services and APIs used by front-end developers and mobile application developers.

Full Stack Development

Get the skills to work with both back-end and front-end technologies as a full-stack developer. You'll develop a solid foundation for working with servers and host configurations, performing database integrations, and troubleshooting front-end development issues.

Project

In this module students work together in a team on a predefined project. The goal of this module is to combine all your acquired skills from the last modules into a project. In this course requirement analysis and project management are core themes while building a full-stack project.
